# Searching kata untuk Sambung Kata

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Pustaka HTML sederhana untuk melakukan pencarian kata atau istilah dalam Kamus Besar Bahasa Indonesia (KBBI) secara terprogram. Proyek ini mempermudah pengembang untuk mengintegrasikan data KBBI ke dalam aplikasi mereka tanpa harus melakukan scraping manual.

## ✨ Fitur Utama

- **Pencarian Kata:** Dapat membantu ketika sedang bermain kata sambung.
- **Pencarian Yang Mudah:** Dapat mencari huruf awal dan akhir.
- **Responsif:** Dapat digunakan/diakses melalui perangkat HandPhone maupun Komputer.
- **Ringan:** Minimal ketergantungan (dependencies), fokus pada kecepatan dan kemudahan penggunaan.

## 🛠️ Teknologi yang Digunakan

- **JavaScript**: Bahasa pemrograman utama.
- **Requests**: Untuk menangani permintaan HTTP ke JSON local.
- **BeautifulSoup4**: Untuk melakukan *parsing* atau ekstraksi data dari struktur HTML KBBI.
